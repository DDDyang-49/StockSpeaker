# MEM_004 数据源防污染

## 东方财富脏数据

API 用 `2147483647`（Int.MAX_VALUE）作为数据缺失 sentinel。所有解析到的数值必须过滤此值并重置为 0，否则 AI 基于脏数据输出荒谬结论。

## 高标股转债污染

`fetchTopStock` 不能直接取第一名。必须扩大候选到 `pz=15`，遍历过滤含「转债/ST/退/N」关键字的标的，取第一个安全 A 股。

## API 默认值语义

行情 API 返回 0 可能表示「无数据」而非「0 手」。必须区分 `== 0`（跳过）和 `> 0`（有效），防止误报。

## 全市场情绪

强制使用 `MarketSentimentFetcher` 抓取东方财富/新浪真实 API，拼接入 AI Prompt。严禁 Mock 数据兜底。
