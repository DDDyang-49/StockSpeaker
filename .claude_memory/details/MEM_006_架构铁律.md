# MEM_006 架构铁律

## 技术栈

- **Kotlin 原生 Android**（Jetpack Compose UI + Foreground Service）
- 禁止 Flet/Flutter/React Native：Flet APK 打包 `flet_audio` 必然崩溃，且无法可靠后台保活
- **TTS 零依赖**：仅用 Android 内置 `TextToSpeech`，支持中文离线
- **网络层**：OkHttp 3 秒超时，行情请求独立子线程
- **存储**：SharedPreferences（AppConfig 持久化）

## 打包

GitHub Actions `build_apk.yml`：解析 `build.gradle.kts` 中的 `versionName` 自动创建 Release。

## 音频焦点

`AUDIOFOCUS_LOSS` 时仅重新请求 `MAY_DUCK` 共存，不停止播报——股票语音是短时金融信息，不应被切歌打断。
