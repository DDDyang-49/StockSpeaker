# MEM_003 播报轨道优先级

## 三层轨道

1. **轨道1 实时异动**（最高优先级，可打断一切）→ 30s 独立冷却
2. **轨道1b 异动平复等待** → 连续 3 轮（6s）无异动才进入复盘
3. **轨道1c 异动跟进** → 简洁播报 + AI 复盘
4. **轨道2 定时常规播报** → 可被异动打断，有推迟机制
5. **轨道3 双AI插播** → 混沌/长间隔时触发资金面深度分析

## 关键规则

- **箱体静默**：|当前价 - 上次播报价| < 0.3% 且量比 < 2.0 → 跳过完整播报，只说「横盘震荡」
- **大单阶梯报警**：当前手数 ≥ 上次报警手数 × 1.5 → 无视冷却立即再报
- **异动冷却独立**：大单、涨速、AI 异动三类各有独立 30s 冷却，互不阻塞
