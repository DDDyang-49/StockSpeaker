# MEM_001 息屏保活机制

## 铁律

**绝对禁止**用 `Handler(Looper.getMainLooper()).postDelayed` 做定时循环。Android Doze 模式会挂起主线程消息队列，15-30分钟后定时必然停摆。

## 正确方案

- 后台 `HandlerThread("StockLoop")` 专做定时，其 Looper 不受 Doze 影响
- 主线程 `uiHandler` 仅负责 UI 更新和 TTS 调用
- `PARTIAL_WAKE_LOCK` + 每轮循环防御性检查 `isHeld` 并 `acquire(10000)`，防止国产 ROM 偷偷释放
- TTS 35 秒超时强制重置：`UtteranceProgressListener.onDone` 息屏后可能永不回调
